Django Rest Client
==================

Drf-kit is django rest framework kit which contains help functions.


Requirements
------------

- Python (2.7, 3.2, 3.3, 3.4, 3.5)
- Django (1.8, 1.9, 1.10)


Installation
------------

Install using pip:

.. code-block:: sh

    pip install drf-kit


Configuration
-------------

Add ``'django_rest_client'`` to your ``INSTALLED_APPS`` of your `settings.py` file.

.. code-block:: python

    INSTALLED_APPS = [
        ...
        'drf-kit',
    ]
